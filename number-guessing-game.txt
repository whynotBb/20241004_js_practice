1. 유저는 숫자를 입력할 수 있다.
    - 레벨 선택 가능
        - 레벨 1: 1~10 / 레벨 2 : 1~50 / 레벨 3 : 1~100
2. 유저가 입력한 숫자가 컴퓨터가 뽑은 숫자보다 작으면 Up! / 크면 Down! / 일치하다면 That’s right이라고 뜨고 게임이 종료된다.
5. 유저는 총 5번의 기회가 있다
6. 게임이 종료되면 버튼은 비활성화된다
7. 리셋버튼을 누르면 게임이 초기화된다
8. 유저가 1~100범위 밖에 숫자를 입력할시에 경고메세지가 뜬다 : 레벨설정
9. 유저가 이미 입력한 값을 또 입력할 시에 경고메세지가 뜬다
10. 반응형 UI : UI 참고 - https://www.abcya.com/games/guess_the_number


1. 레벨 설정 (모달)
    - 범위를 선택 한다  
        - 1: 1~10 / 2 : 1~50 / 3: 1~100
    - 시도 횟수를 선택 한다.
        - 3 / 5 / 10회

2. 게입 시작 : 레벨에 맞게 random 숫자 설정
3. input 에 범위를 벗어난 숫자 입력 시 error message 보여주기 
    - input 하단에 error text
    - 벗어난 숫자는 입력되지 않음. 
        예 ~10 까지 인데 11을 입력하면 에러 메시지 띄우고 10 이 자동으로 입력
4. input 숫자 입력 후 + Go  클릭 시 
    - 이미 시도 했던 값을 다시 입력하면 input 비우고 error message 보여주기
    - input 은 지워준다.
    - 결과 노출 : Up / Down / Success
    - fail 시 Hint 판에 숫자 보여주기
    - Hint 숨기기 가능

5. 게임 종료 (모달)
    - 숫자 맞췄을 때
    - 시도 횟수를 초과하였을 때
    - 모달 띄우고 RESTART 보여줌
    - RESTART 클릭 시 레벨설정 화면을 보여줌

* 코드 리팩토링
    - 기능별로 함수로 뽑아주기
